API Compatibility

The APIs provided with the AEVI SDK are designed to be ‘vendor independent’. As an example, this ensures that an app written for one bank also runs on a device supplied by other banks. This design philosophy can result in some APIs being either unique to, or alternatively, unsupported for specific platforms.
