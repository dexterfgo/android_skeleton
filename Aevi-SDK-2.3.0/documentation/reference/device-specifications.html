Device Specification

Albert is a portable, multifunctional, interactive device powered by the Android operating system. It combines the openness of an app-based platform with the same stringent security standards as current card payment (EFTPOS) terminals. Featuring a 7” touchscreen, it enables businesses to install and use Android business and productivity apps, scan cards and accept payments, all within the same device.
